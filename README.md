# semver-min
minified [semver](https://github.com/npm/node-semver)

## Usage

```html
<script type="module">
 import semver from 'https://unpkg.com/semver-min/esm/index.js';
 console.log(semver.valid('1.2.3')); // '1.2.3'
</script>
```
```html
<script src="https://unpkg.com/semver-min/umd/index.js"></script>
<script>
 console.log(semver.valid('1.2.3')); // '1.2.3'
</script>
```
